Figure - Widescreen

Figures are implemented using normal HTML5 figure elements, except they use zero-height, padding, and a background-image property to achieve nice cropping with minimal markup. Image tags should be used with both the source and the background-image as an inline style in order to make sure the images are recognized by crawlers. This is a good implementation because it allows for the most adaptive resizing.
